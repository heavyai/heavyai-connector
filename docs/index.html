connect

Create a connection to the MapD server, generating a client and session ID.

connect(callback: Function): MapdCon
Parameters
callback (Function) A callback that takes (err, success) as its signature. Returns con singleton if successful.
Returns
MapdCon: Object.
Example

Connect to a MapD server:

var con = new MapdCon()
  .host('localhost')
  .port('8080')
  .dbName('myDatabase')
  .user('foo')
  .password('bar')
  .connect((err, con) => console.log(con.sessionId()));

  // ["om9E9Ujgbhl6wIzWgLENncjWsaXRDYLy"]

disconnect

Disconnect from the server and then clear the client and session values.

disconnect(callback: Function): MapdCon
Parameters
callback (Function) A callback that takes (err, success) as its signature. Returns con singleton if successful.
Returns
MapdCon: Object.
Example

Disconnect from the server:

con.sessionId() // ["om9E9Ujgbhl6wIzWgLENncjWsaXRDYLy"]
con.disconnect((err, con) => console.log(err, con))
con.sessionId() === null;
